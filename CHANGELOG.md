# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [1.0.0] - 2026-06-03

### Added

- **Initial open-source release** 🎉
- paper-reading-zh skill: Chinese deep reading mode with evidence boundary constraints
- efficient-literature-survey skill: batch literature review (10-100 papers)
- Multi-source academic search via MCP: paper-distill-mcp (11 sources), academic-mcp (19 sources), academix
- Local PDF reading via pdf-mcp with search/OCR/table extraction
- arXiv paper search via arxiv-paper-mcp and papers-mcp
- Structured reading output format with citation verification
- MIT License, Code of Conduct, Contributing Guide
- Template MCP configuration (`.mcp.json.example`)
